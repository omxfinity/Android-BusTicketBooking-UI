## 🚌 Voyage Bus – Android Bus Ticket Booking UI

### 📖 Overview
Voyage Bus is a modern Android UI project designed for a **Bus Ticket Booking System**.  
It was developed during the **International Summer Training 2026** at **Allenhouse Business School**, in collaboration with **Cosine Theta Ltd**.  
The project focuses on clean design, intuitive navigation, and modular architecture for future backend integration.

---

### ⚙️ Tech Stack
| Component | Technology |
|------------|-------------|
| Language | Kotlin |
| UI Design | XML Layouts |
| Build System | Gradle KTS |
| IDE | Android Studio Giraffe or later |
| Version Control | Git + GitHub |

---

### 🧩 Features
- Elegant and responsive ticket‑booking interface  
- Dynamic seat‑selection layout  
- Payment‑screen prototype with validation logic  
- Reusable components for future backend integration  
- Optimized for light and dark themes
